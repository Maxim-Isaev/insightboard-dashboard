# insightboard-dashboard

"insightBoard" - демонстрационный fullstack-проект: личный дабоард с аутентификацией JWT, CRUD-таблицей и аналитическими графиками. Цель — показать владение современным стеком (React, TypeScript, Node.js, Express, SQLite, Docker).

Стек технологий:

- "Frontend": React 18, TypeScript, Redux Toolkit, React Router, Axios, SCSS Modules, Vite
- "Backend": Node.js, Express, JWT, bcryptjs, SQLite (или PostgreSQL)
- "DevOps": Docker, docker-compose
- "Testing": Jest, React Testing Library
